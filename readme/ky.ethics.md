## Браузердин сатуучусунун кодулоосу
</На>


CloudFlare Firefox колдонуп жаткандарга артыкчылыктуу мамиле кылат, ал эми Tor-эмес браузердин колдонуучуларына Tor-дан ашыкча мамиле жасашат.
Акысыз javascriptти аткаруудан баш тарткан Tor колдонуучулары дагы касташат.
Бул жеткиликтүүлүк теңсиздиги тармактагы бейтараптыкты бузуу жана бийликти кыянаттык менен пайдалануу.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browdifftbcx.gif)

- Солдо: "Tor Browser", Оң: `Chrome`. Бир эле IP дареги.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browserdiff.jpg)

Сол жакта: `[Tor Browser] Javascript өчүрүлгөн, куки иштетилген '
- Оң: `[Chrome] Javascript иштетилген, куки өчүрүлгөн '

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfsiryoublocked.jpg)

- Tor (Clearnet IP) жок QuteBrowser (кошумча браузер)

## Шайлоочуларды бөгөт коюу
</На>


АКШ штаттарындагы шайлоочулар добуш берүү үчүн каттоодон өтүшүп, жашаган жеринде мамлекеттик катчынын веб-сайты аркылуу катталат.
Республикалык көзөмөлдөгү мамлекеттик катчынын кеңселери Cloudflare аркылуу статс-катчынын веб-сайтына прокси жасоо аркылуу шайлоочуларды кысымга алуу менен алектенишет.
Cloudflare компаниясынын Tor колдонуучуларына кастык мамилеси, анын MITMдин борборлоштурулган глобалдык көзөмөл пункту катары позициясы жана жалпысынан анын зыяндуу ролу
болочок шайлоочуларды каттоону каалабайт. Либералдар, айрыкча, купуялуулукту кабыл алышат. Шайлоочуларды каттоо бланктарында шайлоочунун саясий ыктыяры, жеке дареги, социалдык камсыздандыруу номери жана туулган күнү жөнүндө жашыруун маалымат чогултулат.
Көпчүлүк штаттар ошол маалыматтын бир гана бөлүгүн жалпыга жеткиликтүү кылышат, бирок Cloudflare кимдир-бирөө добуш берүүгө катталып жатканда *** бардык ушул маалыматты көрөт.

Эскерте кетүүчү нерсе, кагаздарды каттоо Cloudflare программасын кыйратпайт, анткени мамлекеттик катчы кызматкерлеринин маалымат каттарын колдонушу мүмкүн
Маалыматтарды киргизүү үчүн Cloudflare вебсайты.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfvotm_01.jpg)
![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfvotm_02.jpg)

- Change.org - добуш чогултуу жана иш-аракеттерди жүргүзүү боюнча белгилүү сайт. "[бардык жерде адамдар үгүт иштерин жүргүзүп, колдоочуларды мобилизациялап, чечимдерди кабыл алуу үчүн чечим кабыл алуучулар менен иштешүүдө.](https://web.archive.org/web/20200206120027/https://www.change.org/about)"
Тилекке каршы, Cloudflare агрессивдүү чыпкасы себебинен көпчүлүк адамдар Change.org сайтын көрө алышпайт. Алар демократиялык процесстен четтетилген петицияга кол коюуга тоскоолдук кылып жатышат. Булутсуз күйгөн платформаны колдонуу, мисалы, [OpenPetition](https://www.openpetition.eu/content/about_us) көйгөйдү жоюуга жардам берет.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/changeorgasn.jpg)
![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/changeorgtor.jpg)

- Cloudflare'нин "[Athenian Project](https://www.cloudflare.com/athenian/)" мамлекеттик жана жергиликтүү шайлоо веб-сайттарында ишканалар деңгээлинде акысыз коргоону сунуштайт. Алар "алардын шайлоочулары шайлоо жөнүндө маалыматты жана шайлоочуларды каттоо_ мүмкүнчүлүгүн ала алышат" дешти, бирок бул жалган, анткени көпчүлүк адамдар сайтты такыр эле көрө алышпайт.

Бүгүнкү күндө, көптөгөн компаниялар [каттоо эсебиңизди жапканыңыздан же алып салгандан кийин, маалыматтарыңызды сактап турушат](https://justdeleteme.xyz/). Көпчүлүк жакшы компаниялар бул жөнүндө алардын купуялык саясатында эскеришет. Cloudflare? Жок.

```
2019-08-05 CloudFlare мага каттоо эсебимди алып салгандыгын ырастады.
2019-10-02 Мен CloudFlare'ден "анткени мен кардармын" деген электрондук кат алдым

Cloudflare "алып салуу" деген сөздү билген эмес. Эгер ал чын эле _removed_ болсо, анда эмне үчүн мурдагы кардар электрондук почта билдирүүсүн алды? Ал ошондой эле Cloudflare'нин купуялуулук саясаты бул жөнүндө айтылбагандыгын айтты.

```
Алардын жаңы купуялык саясаты бир жыл бою маалыматтарды сактоо жөнүндө эч нерсе айткан жок.
