## CloudFlare spams ປະຊາຊົນ
</summar>


Cloudflare ກຳ ລັງສົ່ງອີເມວສະແປມໃຫ້ຜູ້ໃຊ້ທີ່ບໍ່ແມ່ນ Cloudflare.

- ສົ່ງອີເມວຫາຜູ້ຈອງທີ່ໄດ້ເລືອກເຂົ້າໃຊ້ເທົ່ານັ້ນ
- ເມື່ອຜູ້ໃຊ້ເວົ້າວ່າ "ຢຸດ", ຫຼັງຈາກນັ້ນຢຸດສົ່ງອີເມວ

ມັນງ່າຍດາຍແທ້ໆ. ແຕ່ Cloudflare ບໍ່ສົນໃຈ.
Cloudflare ກ່າວວ່າການໃຊ້ບໍລິການຂອງພວກເຂົາ [ສາມາດຢຸດ spammers ຫຼືຜູ້ໂຈມຕີທັງ ໝົດ](https://support.cloudflare.com/hc/en-us/articles/200170066-Will-activating-Cloudflare-stop-all-spammers-or-attackers- ).
ພວກເຮົາສາມາດຢຸດ _Cloudflare spammers_ ໄດ້ແນວໃດໂດຍບໍ່ຕ້ອງໃຊ້ Cloudflare?

## ຜູ້ໃຊ້ Doxxing
</summar>


Cloudflare ມີບັນຫາ [ການລົບກວນຢ່າງຫຼວງຫຼາຍ](https://web.archive.org/web/20171024040313/http://www.businessinsider.com/cloudflare-ceo-suggests-people-who-report-online-abuse-use -fake-names-2017-5).
Cloudflare [ແບ່ງປັນຂໍ້ມູນສ່ວນຕົວ](https://archive.ph/ePdvi) ຂອງຜູ້ທີ່ [ໃຜ](https://twitter.com/ZJemptv/status/898299709634248704) [ຈົ່ມ](https://twitter.com/TinyPirate/ສະຖານະພາບ/554718958176067584) [ກ່ຽວກັບ](https://twitter.com/remembrancermx/status/1010329041235148802) [ເປັນເຈົ້າພາບ](https://twitter.com/Bridaguy/status/915003769280172037) [sites](https://twitter .com/HelloAndrew/ສະຖານະພາບ/897260208845500416). ບາງຄັ້ງພວກເຂົາຂໍໃຫ້ທ່ານຈັດຫາ

## ການ ຈຳ ແນກຜູ້ຂາຍຂອງຕົວທ່ອງເວັບ
</summar>


CloudFlare ໃຫ້ການຮັກສາບຸລິມະສິດແກ່ຜູ້ທີ່ໃຊ້ Firefox ໃນຂະນະທີ່ໃຫ້ການຮັກສາ hostile ກັບຜູ້ໃຊ້ທີ່ບໍ່ແມ່ນ Tor-Browser ຜ່ານ Tor.
ຜູ້ໃຊ້ Tor ຂອງຜູ້ທີ່ປະຕິເສດຢ່າງຖືກຕ້ອງທີ່ຈະໃຊ້ javascript ທີ່ບໍ່ແມ່ນຟຣີກໍ່ໄດ້ຮັບການປິ່ນປົວທີ່ເປັນສັດຕູ.
ຄວາມບໍ່ສະ ເໝີ ພາບໃນການເຂົ້າເຖິງນີ້ແມ່ນການລ່ວງລະເມີດຄວາມເປັນກາງຂອງເຄືອຂ່າຍແລະການສວຍໃຊ້ ອຳ ນາດ.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browdifftbcx.gif)

- ດ້ານຊ້າຍ: `Tor Browser`, ຂວາ:` Chrome`. ທີ່ຢູ່ IP ດຽວກັນ.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browserdiff.jpg)

- ດ້ານຊ້າຍ: `[Tor Browser] Javascript ພິການ, ຄຸກກີເປີດໃຊ້`
- ຖືກຕ້ອງ: `[Chrome] Javascript ເປີດໃຊ້ງານ, ຄຸກກີພິການ`

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfsiryoublocked.jpg)

## ການສະກັດກັ້ນຜູ້ລົງຄະແນນສຽງ
</summar>


ບັນດາຜູ້ມີສິດເລືອກຕັ້ງໃນລັດຕ່າງໆຂອງສະຫະລັດອາເມລິກາລົງທະບຽນລົງຄະແນນສຽງໃນທີ່ສຸດຜ່ານເວບໄຊທ໌ຂອງເລຂາທິການລັດໃນລັດທີ່ຢູ່ອາໄສຂອງພວກເຂົາ.
ຫ້ອງການເລຂານຸການຂອງລັດທີ່ຄວບຄຸມໂດຍພັກ Republican ເຂົ້າຮ່ວມໃນການສະກັດກັ້ນຜູ້ລົງຄະແນນສຽງໂດຍການສົ່ງຕໍ່ເວັບໄຊທ໌ຂອງເລຂາທິການລັດຜ່ານ Cloudflare.
ການຮັກສາ hostile ຂອງ Cloudflare ໃຫ້ກັບຜູ້ໃຊ້ Tor, ຕຳ ແໜ່ງ MITM ຂອງມັນເປັນຈຸດສູນກາງທົ່ວໂລກຂອງການເຝົ້າລະວັງ, ແລະບົດບາດທີ່ ທຳ ລາຍໂດຍລວມ
ເຮັດໃຫ້ຜູ້ມີສິດເລືອກຕັ້ງທີ່ບໍ່ຕັ້ງໃຈຈະລົງທະບຽນ. ເສລີພາບໂດຍສະເພາະມີແນວໂນ້ມທີ່ຈະຮັບເອົາຄວາມເປັນສ່ວນຕົວ. ແບບຟອມລົງທະບຽນຜູ້ລົງຄະແນນສຽງລວບລວມຂໍ້ມູນທີ່ລະອຽດກ່ຽວກັບການຢັ່ງຫາງສຽງທາງດ້ານການເມືອງຂອງຜູ້ລົງຄະແນນ, ທີ່ຢູ່ທາງດ້ານຮ່າງກາຍສ່ວນຕົວ, ເລກປະກັນສັງຄົມແລະວັນເດືອນປີເກີດ.
ລັດສ່ວນໃຫຍ່ພຽງແຕ່ສ້າງຂໍ້ມູນຍ່ອຍນັ້ນອອກສູ່ສາທາລະນະ, ແຕ່ Cloudflare ເຫັນວ່າ *** ຂໍ້ມູນທັງ ໝົດ *** ນັ້ນເມື່ອຜູ້ໃດລົງທະບຽນລົງຄະແນນສຽງ.

ໃຫ້ສັງເກດວ່າການລົງທະບຽນເຈ້ຍບໍ່ຫລີກລ້ຽງ Cloudflare ເພາະວ່າພະນັກງານພະນັກງານເຂົ້າຂໍ້ມູນຂອງລັດຖະບານອາດຈະໃຊ້
ເວັບໄຊທ໌ Cloudflare ເພື່ອປ້ອນຂໍ້ມູນ.
