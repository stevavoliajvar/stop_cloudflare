## CloudFlare skräppost människor
</Summary>


Cloudflare skickar skräppostmeddelanden till användare som inte är Cloudflare.

- Skicka bara e-postmeddelanden till prenumeranter som har valt
- När användaren säger "stopp", sluta sedan skicka e-post

Det är så enkelt. Men Cloudflare bryr sig inte.
Cloudflare sa att man använder sin tjänst [kan stoppa alla spammare eller angripare](https://support.cloudflare.com/hc/en-us/articles/200170066-Will-activating-Cloudflare-stop-all-spammers-or-attackers- ).
Hur kan vi stoppa _Cloudflare-spammare_ utan att aktivera Cloudflare?

## Diskriminering av webbläsarsäljare
</Summary>


CloudFlare ger förmånsbehandling för dem som använder Firefox samtidigt som de använder fientlig behandling till användare av icke-Tor-Browser över Tor.
Tor-användare av som med rätta vägrar att utföra icke-gratis javascript får också fientlig behandling.
Denna ojämlikhet i tillträde är ett missbruk av nätneutralitet och maktmisbruk.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browdifftbcx.gif)

- Vänster: `Tor Browser`, Right:` Chrome`. Samma IP-adress.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browserdiff.jpg)

- Vänster: `[Tor Browser] Javascript Disabled, Cookie Enabled`
- Höger: `[Chrome] Javascript aktiverat, cookie inaktiverat`

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfsiryoublocked.jpg)

- QuteBrowser (mindre webbläsare) utan Tor (Clearnet IP)

| *** Webbläsare *** | *** Tillgångsbehandling *** |
| --- | --- |
| Tor Browser (Javascript aktiverat) | åtkomst tillåten |
| Firefox (Javascript aktiverat) | tillgång försämrad |
| Krom (Javascript aktiverat) | åtkomst degraderad (trycker på Google reCAPTCHA) |
| Krom eller Firefox (Javascript inaktiverat) | åtkomst nekad (skjuter * trasig * Google reCAPTCHA) |
| Chromium eller Firefox (Cookie inaktiverat) | åtkomst nekad |
| QuteBrowser | åtkomst nekad |
| lynx | åtkomst nekad |
| w3m | åtkomst nekad |
| wget | åtkomst nekad |


"_Varför använder du inte Ljudknappen för att lösa enkel utmaning? _"

Ja, det finns en ljudknapp, men den _always_ [fungerar inte över Tor](https://trac.torproject.org/projects/tor/ticket/23840). Du får detta meddelande när du klickar på det:

'' '
Försök igen senare
Din dator eller nätverk skickar automatiskt frågor.
För att skydda våra användare kan vi inte behandla din begäran just nu.
För mer information besök vår hjälp sida
'' '

## Väljarundertryckning
</Summary>


Väljare i amerikanska stater registrerar sig för att i slutändan rösta via statssekreterarens webbplats i deras hemvist.
Republikanska kontrollerade statssekretariatskontor deltar i väljarundertryckning genom att stänga av statssekreterarens webbplats genom Cloudflare.
Cloudflares fientliga behandling av Tor-användare, dess MITM-position som en centraliserad global övervakningspunkt och dess skadliga roll totalt sett
gör potentiella väljare ovilliga att registrera sig. Särskilt liberaler tenderar att omfamna integritet. Väljarregistreringsformulär samlar känslig information om en väljares politiska lutning, personliga fysiska adress, personnummer och födelsedatum.
De flesta stater gör bara en delmängd av den informationen allmänt tillgänglig, men Cloudflare ser *** all *** den informationen när någon registrerar sig för att rösta.

Observera att pappersregistrering inte kringgår Cloudflare eftersom sekreteraren för anställda för anmälningsuppgifter för personuppgifter sannolikt kommer att använda
Cloudflare-webbplats för att ange data.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfvotm_01.jpg)
![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfvotm_02.jpg)

- Change.org är en berömd webbplats för att samla röster och vidta åtgärder. "[människor överallt startar kampanjer, mobiliserar supportrar och arbetar med beslutsfattare för att driva lösningar.](https://web.archive.org/web/20200206120027/https://www.change.org/about)"
Tyvärr kan många inte se Change.org alls på grund av Cloudflares aggressiva filter. De blockeras från att underteckna framställningen, vilket utesluter dem från en demokratisk process. Att använda annan plattform som inte är molnklar, som [OpenPetition](https://www.openpetition.eu/content/about_us) hjälper till att avhjälpa problemet.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/changeorgasn.jpg)
![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/changeorgtor.jpg)

- Cloudflares "[Athenian Project](https://www.cloudflare.com/athenian/)" erbjuder gratis företagsnivåskydd för statliga och lokala valwebbplatser. De sa att "deras valkretsar kan få tillgång till valinformation och väljarregistrering", men det är en lögn eftersom många bara inte kan surfa på webbplatsen alls.

</Summary>

## Ljuga om att radera användarens data
</Summary>


Enligt denna [ex-cloudflare-kundens blogg](https://shkspr.mobi/blog/2019/11/can-you-trust-cloudflare-with-your-personal-data/) ljuger Cloudflare om att ta bort konton. Numera har många [företag behåller dina uppgifter](https://justdeleteme.xyz/) efter att du har stängt eller tagit bort ditt konto. De flesta bra företag nämner om det i sin integritetspolicy. CloudFlare? Nej.

'' '
2019-08-05 CloudFlare skickade en bekräftelse på att de har tagit bort mitt konto.
2019-10-02 Jag fick ett e-postmeddelande från CloudFlare "eftersom jag är kund"
'' '

Cloudflare visste inte om ordet "ta bort". Om det verkligen är _removed_, varför fick denna ex-kund ett e-postmeddelande? Han nämnde också att Cloudflares sekretesspolicy inte nämner om den.

'' '
Deras nya sekretesspolicy nämner inte att bevara data under ett år.
'' '

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfviopl_notdel.jpg)

Hur kan du lita på Cloudflare om [deras integritetspolicy är en LIE](https://twitter.com/daviddlow/status/1197787135526555648)?

</Detaljer>
